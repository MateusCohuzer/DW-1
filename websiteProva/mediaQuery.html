Media Queries

As Media Queries em CSS são formas de guiar o navegador por determinadas características, recursos e preferências, encontrando determinados tipos de navegadores e tamanhos de tela diferentes, para assim aplicar estilos ou executar outro código com base nessas características. As consultas de mídia mais comuns no mundo são aquelas que visam intervalos de janela de visualização específicos para aplicar estilos personalizados, que deram origem a toda a ideia de design responsivo. Por exemplo:

                        //
                        /* Se o navegador tiver 600px para mais */
                        @media screen and (min-width: 600px) {
                          .element {
                            /* Aplica um estilo*/
                          }
                        }
